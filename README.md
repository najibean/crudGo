## membuat toko online dengan Golang (Fullstack App)
Sebagai bahan belajar membuat implementasi CRUD dengan Golang. 
<br>
<br>

### Daftar fitur yang akan dibuat

-   Home Page
-   Register, Login, Profile User
-   Products Page
-   Single Product Page
-   Shopping Cart
-   Checkout
    -  Shipping Cost Calculation
    -  Payment Gateway
-   Order History
-   Admin Panel:
    -   Dashboard
    -   Manage Products
    -   Manage Orders
    -   Manage Users
    -   Reports
    -   Configuration

<br>

### Technology Stack
-   Golang
    -   Gorilla Mux (router)
    -   Gorm (ORM ke database)
    -   ...
-   PostgreSQL (database)
-   Bootstrap (interface)
